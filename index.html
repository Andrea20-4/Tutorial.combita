Que es CSS

CSS (Cascading Style Sheets, o "Hojas de Estilo en Cascada" en español) es un lenguaje utilizado para describir la presentación y el diseño de un documento escrito en HTML o XML. CSS permite definir el estilo visual de las páginas web, como los colores, los márgenes, los bordes, las fuentes y el espaciado, entre otros. En resumen, CSS se encarga de la parte visual de una página web, mientras que HTML se ocupa de la estructura y contenido. Así, HTML define el contenido de la página (como párrafos, encabezados, imágenes y enlaces), y CSS se encarga de cómo ese contenido se verá en la pantalla.

Algunas de las principales funciones de CSS:

  • Establecer colores: Cambiar el color de texto, fondos y bordes. Fuente y tipografía: Definir el tipo, tamaño, peso y estilo de la fuente. Diseño de la disposición: Controlar el diseño de los elementos en la página (por ejemplo, alineación, márgenes, espaciado entre elementos). Posicionamiento de elementos: Colocar los elementos de manera precisa en la página (por ejemplo, en columnas o de forma flotante). Transiciones y animaciones: Agregar efectos visuales, como desvanecerse, deslizarse o mover elementos con transiciones suaves.

Aquí te dejo algunas de las propiedades más comunes que puedes usar para dar formato a los textos:

1. Color del texto Para cambiar el color del texto, se usa la propiedad color. p { color: blue; /* Cambia el color del texto a azul */ }

2. Tamaño del texto Se utiliza la propiedad font-size para establecer el tamaño del texto. h1 { font-size: 36px; /* Cambia el tamaño del texto a 36 píxeles */ }

3. Tipo de fuente Puedes cambiar el tipo de fuente con font-family. body { font-family: Arial, sans-serif; /* Establece la fuente Arial */ }

4. Estilo de la fuente La propiedad font-style permite poner el texto en cursiva. em { font-style: italic; /* Aplica cursiva al texto */ }

5. Peso de la fuente Para hacer el texto más grueso o más delgado, se usa font-weight. strong { font-weight: bold; /* Hace el texto en negrita */ }

6. Altura de la línea Puedes ajustar la distancia entre líneas de texto usando line-height. p { line-height: 1.6; /* Aumenta el espacio entre líneas */ }

7. Transformación del texto La propiedad text-transform permite modificar el caso del texto (mayúsculas, minúsculas, etc.). h2 { text-transform: uppercase; /* Convierte todo el texto en mayúsculas */ }

8. Alineación del texto Usa text-align para alinear el texto (izquierda, centro, derecha). h1 { text-align: center; /* Centra el texto */ }

9. Espaciado entre letras La propiedad letter-spacing te permite aumentar o reducir el espacio entre caracteres. h1 { letter-spacing: 2px; /* Aumenta el espacio entre letras */ }

10. Sombra en el texto Para agregar sombra al texto, se usa text-shadow. h1 { text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3); /* Sombra de texto */ }

11. Decoración del texto Usa text-decoration para agregar subrayado, tachado o una línea sobre el texto. a { text-decoration: underline; /* Subraya el texto */ }
