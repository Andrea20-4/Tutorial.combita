¿Qué es CSS?

CSS (Cascading Style Sheets, o "Hojas de Estilo en Cascada") es un lenguaje que describe la presentación y el diseño de un documento HTML o XML. Permite definir estilos visuales de páginas web, como colores, márgenes, bordes, fuentes y espaciado.

En resumen, HTML define la estructura y contenido de una página, mientras que CSS se encarga de cómo se verá ese contenido.

Algunas de las principales funciones de CSS

  • Establecer colores: Cambiar el color de texto, fondos y bordes.
  • Fuente y tipografía: Definir el tipo, tamaño, peso y estilo de la fuente.
  • Diseño de la disposición: Controlar la ubicación de los elementos (alineación, márgenes, espaciado).
  • Posicionamiento de elementos: Colocar los elementos de forma precisa (columnas, flotantes, posicionamiento absoluto, etc.).
  • Transiciones y animaciones: Agregar efectos visuales como desvanecerse, deslizarse o moverse con transiciones suaves.

Propiedades más comunes para dar formato a los textos

1. Color del texto

p {
  color: blue; /* Cambia el color del texto a azul */
}

Ejemplo de texto en color azul

2. Tamaño del texto

h1 {
  font-size: 36px; /* Cambia el tamaño del texto a 36px */
}

3. Tipo de fuente

body {
  font-family: Arial, sans-serif; /* Establece la fuente Arial */
}

4. Estilo de la fuente

em {
  font-style: italic; /* Aplica cursiva al texto */
}

5. Peso de la fuente

strong {
  font-weight: bold; /* Hace el texto en negrita */
}

6. Altura de la línea

p {
  line-height: 1.6; /* Aumenta el espacio entre líneas */
}

7. Transformación del texto

h2 {
  text-transform: uppercase; /* Convierte el texto a mayúsculas */
}

8. Alineación del texto

h1 {
  text-align: center; /* Centra el texto */
}

9. Espaciado entre letras

h1 {
  letter-spacing: 2px; /* Aumenta el espacio entre letras */
}

10. Sombra en el texto

h1 {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3); /* Sombra suave */
}

11. Decoración del texto

a {
  text-decoration: underline; /* Subraya el texto */
}

¡Experimenta con estas propiedades para dar estilo a tu contenido!
